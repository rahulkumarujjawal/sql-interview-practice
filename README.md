# SQL Interview Practice Repository

This repository contains structured SQL interview problems categorized into Basic, Challenging, and Advanced levels.

## Topics Covered

- Aggregations
- Window Functions
- Subqueries
- CTE
- Recursive Queries
- Ranking Functions
- Date Calculations
- Data Cleaning Techniques

Each file contains:
- Problem Statement
- Assumed Table Structure
- SQL Solution
- Explanation

The goal is to strengthen analytical and problem-solving skills for real-world SQL interviews.
